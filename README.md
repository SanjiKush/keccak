# Keccak

SHA3-256 Implem.

Tests are verified with the official hashlib, https://docs.python.org/3/library/hashlib.html given by KeccakTeam

Usage : 

Linux users:

```bash
make install
make test
```

Others:

```bash
python -m pip install requirements.txt
python test.py
python keccak.py <stringToHash>
```
